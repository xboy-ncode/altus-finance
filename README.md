## ✨ Core Features

*   **Smart Dashboard**: Real-time overview of your balances, upcoming bills, and dynamic charts for income vs. expenses.
*   **Transactions Management**: Beautiful, filterable, and sortable transaction tables with categories and intelligent tagging.
*   **Wishlists & Savings**: Visual goal tracking with radial progress bars and smart time-to-completion calculations.
*   **Travel Planner**: Dedicated modules for tracking trip budgets. See exactly how much you spent on flights, hotels, and food.
*   **AI Financial Assistant**: A sleek, interactive chat interface to get insights about your spending patterns and predictions.
*   **Full i18n Support**: Native language swapping instantly changes currencies, dates, and localized strings.
*   **Theme Engine**: 6 curated premium color palettes (Oceanic, Midnight, Emerald, Professional, Rose, Sunset) that change the entire app's look and feel without a browser refresh.

## 🛠 Tech Stack

Built with cutting-edge tools to ensure a blazing-fast user experience:

*   **[Next.js 15](https://nextjs.org/)** (App Router, Turbopack)
*   **[React 19](https://react.dev/)**
*   **[Tailwind CSS v4](https://tailwindcss.com/)**
*   **[shadcn/ui](https://ui.shadcn.com/)** (Headless UI components)
*   **[Framer Motion](https://www.framer.com/motion/)** (Fluid animations and transitions)
*   **[i18next](https://www.i18next.com/)** (Robust internationalization)
*   **[Recharts](https://recharts.org/)** (Responsive SVG data visualizations)

## 🚀 Quick Start

Getting started with Altus is incredibly straightforward. You only need Node.js installed.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/finance-trak.git
   cd finance-trak
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) and explore your new dashboard!
